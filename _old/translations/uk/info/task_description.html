Речовий модуль Стефана зламався. Цей модуль відповідав за вимову чисел. Для нього зараз велика проблема вимовляти складені числа. Допоможіть нашому Роботу заговорити правильно і освоїти хоча б першу тисячу. Стефан повинен говорити англійською, так що вам потрібно знати правила складання чисел на англійській мові. Всі слова в строковому поданні числа повинні бути розділені одним пропуском. Будьте обережні з пробілами - дуже складно побачити подвійний пробіл, але це критично для комп'ютера.

Вхідні дані: Число у вигляді цілого числа (int).

Вихідні дані: Текстове написання числа, як рядок (str).

Приклади: 

checkio(4)=='four'
checkio(143)=='one hundred forty three'
checkio(12)=='twelve'
checkio(101)=='one hundred one'
checkio(212)=='two hundred twelve'
checkio(40)=='forty'

Як це використовується: Ця концепція буде корисна для програмного забезпечення з синтезу мови або для автоматичних систем звітності. Також це може стати в нагоді при написанні простого бота для чату, який буде вміти складати числа.

Передумови: 0 < number < 1000
